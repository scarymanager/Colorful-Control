# Colorful Control

Colorful Control is a planned cross-platform utility for controlling RGB
keyboards on Colorful and Clevo laptops.

The project will target Linux first, with an architecture that can later
support Windows. It is currently a scaffold only: no keyboard, GUI, backend,
animation, or business functionality has been implemented.

## Development

The project requires Python 3.12 or later. Tooling configuration for pytest
and Ruff is provided in `pyproject.toml`.
